Toadstool

A scrolling platform game made by Maggie Yang for the CPE 471 milestone project at Cal Poly State University in San Luis Obispo, CA.

California Polytechnic State University, San Luis Obispo, 2021.

myang57@calpoly.edu

About


Description

A front-scrolling game where the user controls a frog that has to jump on top of mushrooms until it reaches the coveted water. The game will reset to the beginning if the frog's position is behind the camera or if it falls to the ground. The game ends when the frog reaches the water and can finally enjoy the view!


Controls

Press P to start the game and to be able to control the frog.
The frog can only control its direction once it is in the air. Press SPACE to jump, and WASD to control the direction.


Concepts Utilized

    Collision Detection

    Infinite Terrain & Skysphere

    Diffuse / Specular Lighting

    Texture / Normal / Height Mapping

    Hierarchical Modeling

    Billboard Animation

    Post-processing: Gamma Correction

Architecture


OpenGL

The game was written using the OpenGL API. It also used GLFW and GLM.
